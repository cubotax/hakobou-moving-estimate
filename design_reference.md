# 参考サイト デザイン分析

## サイト情報
- URL: https://singlemove-jjykrpet.manus.space
- ターゲット: 10-20代向け単身引越しサービス

## デザイン特徴

### カラーパレット
- 背景: 白（#FFFFFF）
- プライマリアクセント: 鮮やかな黄色（イエロー）- CTAボタン、ハイライト
- セカンダリアクセント: ピンク（バッジ「学割あり！」）、グリーン（バッジ「スピード対応」）
- テキスト: 黒（#000000）
- ボーダー: 黒の太い線（手書き風）

### タイポグラフィ
- 見出し: 太字、大きめ、手書き風のアンダーライン装飾
- 本文: シンプルで読みやすい
- 日本語フォント: 丸ゴシック系または手書き風

### レイアウト・スタイル
- カード: 黒い太枠ボーダー（3-4px）、角丸
- バッジ: 鮮やかな色（黄色、ピンク、グリーン）で右上に配置
- ボタン: 黄色背景、黒テキスト、太枠ボーダー
- 全体的にポップでカジュアルな印象
- 手書き風の装飾要素（アンダーライン、矢印など）

### UI要素
- ステップ表示: 番号付きの丸いバッジ
- 料金プラン: カード形式、人気プランは黄色背景でハイライト
- FAQ: アコーディオン形式

## 適用方針
1. 背景を白ベースに変更
2. プライマリカラーを黄色に変更
3. カードに黒い太枠ボーダーを追加
4. バッジをカラフルに
5. ボタンを黄色＋黒枠に
6. フォントをより親しみやすいものに
